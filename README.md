# Agenda de Tarefas Python
**Versão:** 1.0.0

## Problema Real
A sobrecarga de informações gera ansiedade e esquecimento. Esta aplicação ajuda estudantes e profissionais a centralizarem suas rotinas de forma rápida pelo terminal, ajudando na organização do dia a dia.

## Como executar
1. Tenha o Python instalado.
2. Clone este repositório.
3. No terminal, instale as dependências: `pip install -r requirements.txt`
4. Execute a aplicação: `python projetoB2.py`

## Como testar
- Para rodar os testes: `pytest`
- Para verificação de qualidade do código (linting): `flake8 .`
